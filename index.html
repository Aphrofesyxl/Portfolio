Habilidades em Design

Illustrator Intermediário

preenchimento e contorno simples, seleção e transformação, uso básico do pathfinder,
x manipulação de imagens, compreensão de gradientes básicos, trabalho com máscaras básicas
e exportação de arquivos.

Photoshop Básico

Corte e redimensionamento, ajuste de cor e brilho, ferramenta de pincel e borracha,
seleção simples, camadas simples, texto simples, remoção de fundo, filtros básicos e
salvamento e exportação de arquivo.

After Effects Básico

Compreensão da interface, importação de mídia, organização das camadas,
animação básica de propriedades, uso de ferramentas de texto, aplicação de efeitos básicos,
trabalho com keyframes, uso de máscaras simples, transição básica e exportação de projeto.

Figma Intermediário

Design e layouts responsivos, componentes e estilos, compartilhamento e colaboração,

Habilidades em Programação

HTML Intermediáio


Estrutura Básica: Conhecimento da estrutura básica de um documento HTML com as tags html, head, e body.

Elementos e Tags: Familiaridade com tags HTML essenciais, como p para parágrafos, h1 a h6 para títulos,
a para links, img para imagens, ul e li para listas não ordenadas, ol e li para listas ordenadas, etc.

Atributos: Compreensão de atributos básicos, como src para especificar a fonte de uma imagem, href para links,
e alt para texto alternativo de imagens.

Links e Navegação: Criação de links internos e externos para navegação entre páginas.

Formulários Simples: Construção de formulários básicos com tags como form, input, label, e button.

Uso de Imagens: Incorporação de imagens usando a tag img.

Listas e Tabelas Básicas: Criação de listas e tabelas simples para organizar conteúdo.

Comentários: Utilização de comentários HTML para adicionar notas explicativas no código.

Validação Básica: Compreensão das práticas básicas de validação de HTML,
como garantir que as tags estejam devidamente aninhadas e fechadas.

Conceitos de Bloco e Linha: Entendimento de elementos de bloco e de linha e como eles afetam o layout.

Cabeçalhos e Metadados: Uso de tags head para adicionar metadados à página, como o título.

Semântica Básica: Conhecimento básico sobre a semântica de alguns elementos, como header, footer, section

CSS Intermediário


Box Model Avançado: Compreensão aprofundada do box model, incluindo margens, bordas,
preenchimentos e dimensões. Uso eficaz de box-sizing para controlar o dimensionamento dos elementos.

Positioning e Layout: Utilização de position para posicionar elementos de forma absoluta, relativa, fixa ou estática.
Uso de técnicas de layout, como float, flexbox e grid.

Estilização de Texto: Aplicação de estilos avançados para texto, como text-transform,
text-shadow e letter-spacing. Uso de fontes externas e personalização de fontes com @font-face.

Estilização de Links: Personalização de estilos para links, incluindo pseudoclasses como :hover e :active.

Transformações e Transições: Aplicação de transformações 2D e 3D, como rotação, escala e inclinação.
Criação de transições suaves usando transition.

Animações CSS: Criação de animações complexas usando a propriedade @keyframes.

Media Queries: Implementação de consultas de mídia para criar layouts responsivos e adaptáveis
a diferentes dispositivos e tamanhos de tela.

Pseudo-elementos e Pseudo-classes: Uso de pseudo-elementos como ::before e ::after.
Utilização de pseudo-classes como :nth-child

Gradients e Sombras: Criação de gradientes usando linear-gradient e radial-gradient.
Aplicação de sombras usando box-shadow e text-shadow.

Flexibilidade e Responsividade: Utilização de flexbox para criar layouts flexíveis.
Implementação de layouts responsivos usando media queries


JavaScript Intermediáio


Manipulação do DOM: Criação dinâmica de elementos HTML.
Adição e remoção de elementos no DOM em resposta a eventos.

Event Listeners: Utilização do método addEventListener para lidar com eventos,
como cliques em elementos específicos.

Lógica Condicional: Implementação de lógica condicional para verificar a visibilidade de elementos na tela.
Aplicação de classes com base em condições específicas.

Iteração e Manipulação de Coleções: Utilização eficiente do método forEach
para iterar sobre elementos em coleções.

Criação de Elementos Dinâmicos: Habilidade para criar dinamicamente elementos HTML, como modais,
em resposta a interações do usuário. Criação de transições suaves usando transition.

Toggle de Classes: Manipulação eficaz de classes para alternar estados de visibilidade e aplicar estilos dinâmicos.
Utilização de classes para criar animações simples.

Manipulação de Atributos: Modificação dinâmica de atributos, como src e alt, para controle de conteúdo.

Animação: Tentativa de animação com a adição/remoção da classe .show-animate.
Exploração de técnicas para criar transições visuais.
